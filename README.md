# SeRuput_Teh
CRUD Application using Java Programming Language (JavaFX) and XAMPP 

Requirements for running this project:
1. Java 11.0.18
2. JavaFX 17.0.7
3. MySQL Java Connection Library 8.0.24
4. XAMPP 8.0.7
